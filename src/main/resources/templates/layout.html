Te presento la aplicación web para clínicas veterinarias.

Diseñada para optimizar la gestión de tu clínica, te permite almacenar y organizar fácilmente los datos tanto de dueños como de sus mascotas.

Podrás vincular a cada dueño con múltiples mascotas y registrar cada consulta o atención médica realizada, generando una completa historia clínica que podrás consultar, editar e incluso imprimir en cualquier momento.

Con una interfaz intuitiva y funcionalidades avanzadas, nuestro sistema está desarrollado con las mejores tecnologías: Java Spring Boot, MySQL, Thymeleaf y Bootstrap, bajo un robusto modelo multicapa que asegura seguridad y rendimiento.
